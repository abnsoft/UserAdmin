Проект выполнен по ТЗ.

1. Проекту требуетя выполнить настройку коннекта к БД : 
    
    Настройка конекта к БД MySQL 
    
    конифиг-файлы к БД лежат : \src\webapp\WEB-INF\spring\db.config
    
    1.1 нужно изменить на ваши параметры конекта к БД 

        db1.jdbc.url=jdbc:mysql://host:3306/DEVABN01
        db1.jdbc.username=USER_MYSQL
        db1.jdbc.password=PASSWORD

    1.2 дамп БД лежит в корне в файле DEVABN01.sql - или можно изменить параметр в файле см .п.2.1 hibernate.hbm2ddl.auto=validate на create, затем вернуть на validate/ или закомментировать данную опцию.

2. Для сборки проекта требуется Maven (использовалась версия 3.2.5), JDK 1.7
    проект собирается в WAR запуском команды из консоли в корне проекта: 
    mvn package 


3. 
- проект выполнен с использованием IDE Eclipse LUNA, Dreamveawer CS6.
- в проекте исползованы следующи библиотеки и технологии : SPRING (-webmvc, -security, -data-jpa), Hibernate JPA, JACKSON, JSON, REST, JQUery, JavaScript, Html.
- авторизация (доступ в админку) выполнен на основе spring-security + jdbc/db
